# AndroidBluetoothApp
This Android proyect has been done  having in mind the target of creating an app
to control the Arduino Device (using the HC-05 BT module) wich rules the robot(car) im already designing.
If anyone is interested in modifying the code for a similar purpose, I asume, it will be
easy because the code is well structured and commented.

The joystick class I used belongs, or at least i took it from him, to Ake Exorcist(Youtube Channel).
You can find his own article in http://www.akexorcist.com/2012/10/android-code-joystick-controller.html

Hope it will be useful for you guys!



Este proyecto de Arduino ha sido llevado a cabo con el fin de disponer de una aplicación  customizada para un
coche-robot(integrado por un Arduino Micro y un módulo bluetooth HC-05) que se encuentra en fase de diseño e implementación .
Si alguien está interesado en modificar el código para un proyecto de características similares no le resultará 
demasiado complicado ya que se encuentra bien comentado y estructurado.

La clase empleada para la utilización de un joystic se ha tomado del Youtuber Ake Exorcist.
Podéis acceder a su artículo en http://www.akexorcist.com/2012/10/android-code-joystick-controller.html 

Espero que os resulte de utilidad!!
